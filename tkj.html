Gambaran

Apa itu TKJ?

TKJ mempelajari perangkat komputer dan jaringan: mulai dari perangkat keras, sistem operasi, pengkabelan, konfigurasi perangkat jaringan (router/switch), sampai perawatan dan perbaikan ketika jaringan bermasalah.

Belajar dari skenario dunia kerja

Kamu akan sering dapat studi kasus: “kenapa internet putus?”, “kenapa VLAN tidak jalan?”, “kenapa server tidak bisa diakses?”— lalu kamu analisis dan perbaiki seperti teknisi sungguhan.

Skill umum TKJ mencakup konfigurasi perangkat jaringan (contoh: Mikrotik/Cisco), Linux/Windows server dasar, dan troubleshooting.

Perangkat lab (router/switch/rack) untuk latihan routing, switching, dan manajemen kabel.

Kompetensi

Yang kamu pelajari

Rangkuman materi TKJ berikut dibuat berdasarkan gambaran umum kurikulum TKJ di SMK (jaringan, OS, perangkat, dan troubleshooting).

Komputer & Sistem Operasi

  • Perakitan, perawatan, dan perbaikan PC
  • Instalasi OS (Windows/Linux) & driver
  • Manajemen user, file sharing, permission dasar
  • Tools diagnosis (resource monitor, log, dsb.)
  • Keamanan dasar perangkat (hardening ringan)

Jaringan Dasar–Menengah

  • Pengenalan topologi, IP addressing, subnetting
  • Konfigurasi LAN (switching, VLAN dasar)
  • Routing dasar (static route), NAT dasar
  • Wireless dasar (SSID, security, channel)
  • Troubleshooting (ping, traceroute, log, kabel)

Server & Layanan Jaringan

  • DHCP/DNS dasar
  • Web server dasar (pengenalan)
  • Remote access & manajemen perangkat
  • Backup & dokumentasi konfigurasi
  • Keamanan jaringan dasar (firewall rules sederhana)

Keunggulan

Kenapa memilih TKJ?

Skill-nya praktikal dan langsung “kepakai” di banyak instansi/industri.

Praktik dominan

Belajar lewat konfigurasi dan troubleshooting, bukan teori saja.

Peluang kerja luas

Teknisi jaringan, IT support, admin jaringan/sistem.

Bisa wirausaha

Service komputer, instalasi jaringan, setup warnet/kantor.

Skill bertahap

Dari LAN → server → network security dasar.

Output

Apa yang kamu dapat dari jurusan ini?

Biar kebayang hasil akhirnya setelah latihan rutin.

Hard skills

  • Merakit & troubleshooting PC
  • Setup LAN + VLAN dasar
  • Konfigurasi routing/NAT dasar
  • Setup Wi-Fi & keamanan dasar
  • Server services dasar (DHCP/DNS, sharing)
  • Dokumentasi topologi & konfigurasi

Soft skills

  • Teliti & rapi (cable management & dokumentasi)
  • Problem solving cepat saat jaringan down
  • Komunikasi teknis (jelasin masalah ke user)
  • Kerja tim (support & maintenance)
  • Disiplin SOP & keselamatan kerja

Bonus: kamu bisa mulai bikin “home lab” mini untuk latihan mandiri.
