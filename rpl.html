Apa itu RPL?

RPL (Rekayasa Perangkat Lunak) adalah bidang yang mempelajari proses merancang, membangun, menguji, dan merilis perangkat lunak. Fokusnya bukan hanya “ngoding”, tapi juga cara berpikir sistematis, struktur data, database, dan membangun aplikasi yang rapi serta mudah dikembangkan.

Belajar lewat proyek nyata

Kamu akan terbiasa kerja seperti mini-tim dev: menyusun kebutuhan fitur, membuat database, membangun UI, menghubungkan ke data, testing, sampai presentasi hasil.

Output-nya bisa kamu jadikan portofolio untuk PKL, magang, lomba, atau freelance.

Suasana lab: tempat latihan coding, kolaborasi, dan demo project.

Keunggulan

Kenapa memilih RPL?

Skill-nya fleksibel: bisa masuk industri software, bikin produk sendiri, atau lanjut kuliah.

Portofolio jelas

Web/app/database yang bisa didemokan jadi nilai plus.

Problem solving

Terbiasa berpikir logis, rapi, dan terstruktur.

Skill dicari

Web dev, database, OOP, integrasi sistem.

Bisa wirausaha

Siap bikin web toko online, sistem sekolah, atau aplikasi UMKM.

Output

Apa yang kamu dapat dari jurusan ini?

Ringkas tapi detail biar kebayang hasil akhirnya.

Hard skills

  • Algoritma dasar/lanjut + OOP
  • Database (desain, query SQL, integrasi)
  • Web statis/dinamis + aplikasi web berbasis objek
  • Aplikasi desktop (visual / paket software)
  • Dasar OS & jaringan untuk kebutuhan deployment
  • Basic e-commerce (muatan lokal)

Soft skills

  • Berpikir sistematis & teliti (debugging)
  • Manajemen waktu (deadline project)

  • Problem solving (error, logic, integrasi)
